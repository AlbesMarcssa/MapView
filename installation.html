Using MapView in an Android Project (Eclipse)

  1. Create a folder in your application root (next to bin, src, etc.) named "libs"
  2. Download the .jar file to this folder.
  3. In Eclipse, right click on the .jar file and select "Build Path" > "Add to Build Path"
  4. That's it. You can now instantiate normally: new MapView( context );

This widget uses the LruCache for bitmap caching. If you're targeting API less than 12, you'll need to do the same process (adding the jar to your project's build path) with the Android Support Library.

Jake Wharton's DiskLruCache is also used, but is included in the mapviewlib.jar linked in step 1, so no extra steps are necessary.
